# Nimble Gravity | React + TypeScript + Vite

## Setup

1.  **Install dependencies**:
    ```bash
    npm install
    ```

2.  **Environment Variables**:
    Copy `.env.example` to `.env` and modify with your details:
    ```bash
    cp .env.example .env
    ```
    
    env variables:
    - `VITE_API_BASE_URL`: The API URL provided in the challenge.
    - `VITE_CANDIDATE_EMAIL`: Candidate email.

3.  **Run the app**:
    ```bash
    npm run dev
    ```
